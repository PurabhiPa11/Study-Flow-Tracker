# Study Flow Tracker

Study Flow Tracker is a modern productivity dashboard designed to help students plan, track, and complete their daily study goals.

Built using HTML, CSS, and JavaScript, the application provides an interactive interface for task management, progress tracking, and daily streak monitoring.

---

## Features

- Login screen with personalized greeting
- Task creation with multiple categories
- Circular progress tracker based on completed study hours
- Daily streak system
- Task board for categorized workflow
- Weekly streak calendar
- Task completion and deletion
- Smooth animations and transitions
- Responsive and modern dashboard UI

---

## Categories

The application supports multiple study task types:

- Study
- Coding
- Assignment
- Homework
- Presentation

## How It Works

1. Enter your name and standard/degree
2. Start the dashboard
3. Add tasks with estimated study hours
4. Mark tasks as completed
5. Track progress and streak
